Detailed Description

A collection of GFLineStrings.

MultiLineString can be used to group lines belonging to each other, e.g. a highway (with interruptions)

Tony Stone

- (instancetype) initWithGeoJSONGeometry: (NSDictionary *)  jsonDictionary

Initialize this geometry with the given jsonDictionary.

Note

You must pass the geometry portion of the GeoJSON structure and not the entire GeoJSON object.

Example:

{
"type": "Feature",
"geometry": { "type": "MultiLineString",
"coordinates": [
[ [100.0, 0.0], [101.0, 1.0] ],
[ [102.0, 2.0], [103.0, 3.0] ]
]
}
}

In the above example only the dictionary below that represents the geometry portion is passed.

"type": "MultiLineString",
"coordinates": [
[ [100.0, 0.0], [101.0, 1.0] ],

}
