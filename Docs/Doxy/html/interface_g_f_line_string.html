Method Documentation

- (instancetype) initWithWKT: (NSString *)  wkt

Initialize this geometry with the given WKT (Well-Known-Text) string.

Example:

{
NSString * wkt = @"LINESTRING(40 60,120 110)";
GFLineString * lineString = [[GFLineString alloc] initWithWKT: wkt]];
}

Reimplemented in GFRing.

- (instancetype) initWithGeoJSONGeometry: (NSDictionary *)  jsonDictionary

Initialize this geometry with the given jsonDictionary.

Note

You must pass the geometry portion of the GeoJSON structure and not the entire GeoJSON object.

Example:

{
"type": "Feature",
"geometry": { "type": "LineString",
"coordinates": [ [100.0, 0.0], [101.0, 1.0] ]
}
}

In the above example only the dictionary below that represents the geometry portion is passed.

{
"type": "LineString",
"coordinates": [ [100.0, 0.0], [101.0, 1.0] ]
}

Reimplemented in GFRing.
