Detailed Description

GFBox represents a simple box geometry.

The GFBox class represents a simple box geometry and is represented as min and max corner GFPoints.

- (id) initWithWKT: (NSString *)  wkt

Initialize this geometry with the given WKT (Well-Known-Text) string.

Note

WKT does not support boxes. However, to be generic GeoFeatures supports reading and writing from and to boxes. Boxes are outputted as a standard POLYGON WKT.

GeoFeatures can read boxes from a standard POLYGON, from a POLYGON with 2 points or from a BOX

The WKT output will always be a POLYGON

Example:

{
NSString * wkt = @"POLYGON((0 0,0 90,90 90,90 0,0 0))";
GFBox * box = [[GFBox alloc] initWithWKT: wkt]];
}
- (id) initWithGeoJSONGeometry: (NSDictionary *)  jsonDictionary

Initialize this geometry with the given jsonDictionary.

Note

You must pass the geometry portion of the GeoJSON structure and not the entire GeoJSON object.

Example:

{
"type": "Feature",
"geometry": {
"type": "Box",
"coordinates": [[100.0, 0.0], [101.0, 1.0]]
}
}

In the above example only the dictionary below that represents the geometry portion is passed.

{
"type": "Box",
"coordinates": [[100.0, 0.0], [101.0, 1.0]]
}
