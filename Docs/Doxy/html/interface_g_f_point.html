- (id) initWithWKT: (NSString *)  wkt

Initialize this geometry with the given WKT (Well-Known-Text) string.

Example:

{
NSString * wkt = @"POINT(1 1)";
GFPoint * point = [[GFPoint alloc] initWithWKT: wkt]];
}
- (id) initWithGeoJSONGeometry: (NSDictionary *)  jsonDictionary

Initialize this geometry with the given jsonDictionary.

Note

You must pass the geometry portion of the GeoJSON structure and not the entire GeoJSON object.

Example:

{
"type": "Feature",
"geometry": { "type": "Point",
"coordinates": [100.0, 0.0]
}
}

In the above example only the dictionary below that represents the geometry portion is passed.

{
"type": "Point",
"coordinates": [100.0, 0.0]
}
