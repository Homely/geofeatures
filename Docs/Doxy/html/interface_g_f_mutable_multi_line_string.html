- (void) setObject: (GFLineString *)  aLineString
atIndexedSubscript: (NSUInteger)  index 

Sets the GFLineString at the given index.

Parameters
aLineStringThe GFLineString with which to replace the GFLineString at index in the GFMutableMultiLineString. This value must not be nil.
indexThe index of the GFLineString to be replaced. This value must not exceed the bounds of the GFMutableMultiLineString.

Example:

{
GFMutableMultiLineString * multiLineString = [[GFMutableMultiLineString alloc] init];
multiLineString[0] = [[GFLineString alloc] initWithWKT: @"LINESTRING(1 1,2 2,3 3)"];
multiLineString[1] = [[GFLineString alloc] initWithWKT: @"LINESTRING(2 2,103 1)"];
}
Exceptions
NSExceptionIf index is beyond the end of the collection (that is, if index is greater than or equal to the value returned by count), an NSRangeException is raised.
